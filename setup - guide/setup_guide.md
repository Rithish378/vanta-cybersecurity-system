# VANTA Setup Guide

## Overview
VANTA is a DNS monitoring system built on top of Pi-hole. It collects DNS activity, generates simple visual reports, and sends updates to Telegram.

---

## Requirements

- Raspberry Pi or any Linux system
- Pi-hole installed and running
- Python 3 installed
- Internet connection

---
